Heartbleed

Heartbleed is an exploit that exists in older versions of OpenSSL. Essentially, it is possible to ping a server using heartbeat requests and receiving leaked memory data from the server.

💻 Our contribution

This is a sample login page that sends a GET request to /index.html with the username and password as query string parameters. The request is routed through the server, and can be seen by using the Heartbleed exploit.
